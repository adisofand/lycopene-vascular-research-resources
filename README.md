# lycopene-vascular-research-resources

A curated collection of open-source tools, datasets, and references for researching the effects of lycopene on cardiovascular health, with a focus on vascular function mechanisms.

## Data Analysis Tools

### Rtery
- **Repository**: [jcherubini/Rtery](https://github.com/jcherubini/Rtery)
- **Description**: A suite of tools for the analysis of endothelial function, useful for evaluating vascular health endpoints in lycopene studies.

## Dietary Data Processing

### Nutrition Analysis Notebook
- **Repository**: [Jasc94/projects](https://github.com/Jasc94/projects)
- **Description**: A Jupyter notebook that loads and processes dietary nutrient data (including lycopene content) from the USDA Food and Nutrient Database for Dietary Studies (FNDDS), facilitating antioxidant intake calculations.
